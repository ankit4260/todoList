# 📝 Todo List App

A simple full-stack Todo List app with support for multiple custom lists.

## Tech Stack

- **Backend** — Node.js, Express.js
- **Database** — MongoDB Atlas (Mongoose)
- **Frontend** — EJS, CSS, Vanilla JS
